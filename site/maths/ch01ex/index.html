    Practice Exercise¶

    1. If \(a\in R\) then multiplicative inverse of \(a\) is

      Ans: \(1/a\)

    2. Which of the following is field?

      Ans: \(Q\)

    3. \(1>-1\implies -3>-5\), this property is called

      Ans: Additive property

    4. Let \(a\), \(b\), \(c\), \(d\) \(\in R\) then \(a=b\) and \(c=d\implies\)

      Ans: \(a+c=b+d\)

    5. If \(Z_1=(x_1,y_1)\), \(Z_2=(x_2,y_2)\) where \(Z_1\cdotp Z_2\in C\) then \(Z_1+Z_2=\)?

      Ans: \((x_1+x_2,y_1+y_2)\)

    6. If \(Z_1=3-6i\) and \(Z_2=4+5i\) then \(Z_1Z_2=\)?

      Ans: \(42-9i\)

    7. If \(Z_1=(1,0)\), \(Z_2=(2,3)\) then \(Z_1Z_2=\)?

      Ans: \(2+3i\)

    8. \((3,5)+(0,4)=\)?

      Ans: \((3,9)\)

    9. Additive inverse of \((3,3)\) is \(C\) is

      Ans: \((-3,-3)\)

    10. If \(Z=(a,b)\) then multiplication inverse of \(Z\) is

      Ans: $$ \left(\frac{a}{ a^2 + b^2 }, \frac{-b}{ a^2 + b^2 }\right) $$

    11. If \(Z=a+b\), then \(|Z|=\)?

      Ans: \(a-b\)

    12. If \(Z=a+b\), then \(\overline{Z}=\)?

      Ans: \(a+b\)

    13. \(i^{15}=\)?

      Ans: \(-i\)

    14. The value of \(i^{25}=\)?

      Ans: \(i\)

    15. Conjugate of \((-3,4)\) is

      Ans: \((-3,-4)\)

    16. \((-i)^{31}\)

      Ans: \(i\)

    17. The multiplicative inverse of \(1-3i\) is

      Ans: none

    18. The value of \(i^{-29}\) is

      Ans: \(-i\)

    19. If \(Z\in C\) then \(Z\overline{Z}=\)?

      Ans: \(\overline{Z_1-Z_2}\)

    20. If \(Z_1,Z_2\in C\) then \(\overline{Z_1+Z_2}=\)?

      Ans: \(\overline{Z_1}+\overline{Z_2}\)

    21. The value of \((3+2i)^3\) is

      Ans: \(46i-9\)

    22. \(C\) has no identity with respect to \(+\) other than

      Ans: \(0+0i\)

    23. The conjugate of $$ \frac{2+3i}{1-i} $$

      Ans: $$ \frac{2-3i}{1+i} $$

    24. \((-1+\sqrt{-3})^4+\) \((-1-\sqrt{-3})^4=\)?

      Ans: \(-16\)

    25. The smallest positive \(k\) for which $$ \left(\frac{1+i}{1-i}\right)^k = 1 $$ is

      Ans: \(4\)

    26. Set of even prime natural numbers is

      Ans: Singleton

    27. If \(n\) is any positive integer then the value of $$ \frac{i^{4n+1} - i^{4n-1} }{2}= $$ ?

      Ans: \(i\)

    28. $$ \frac{\sqrt{18}}{\sqrt{72}} $$ is

      Ans: Rational number

    29. \(2\sqrt{-9}\times\sqrt{-16}=\)?

      Ans: \(-24\)

    30. $$ \left\vert\frac{1+i}{a+\frac{1}{i}}\right\vert= $$ ?

      Ans: \(1\)

    31. $$ \left(\frac{1-i}{1+i}\right)^{100} =x+iy $$ then

      Ans: \(x=1\), \(y=0\)

    32. \(\{1,-1\}\) is closed with respect to

      Ans: Both Multiplication & Division

    33. \((\cos20+i\sin20)^5\) \(/(\cos30+i\sin30)^3=\)?

      Ans: \(\cos10+i\sin10\)

    34. The polar coordinates of point are \((2, 319)\) then the Cartesian coordinates of point are

      Ans: \((1.509, -1.312)\)

    35. The set of rational numbers between two real numbers is

      Ans: Infinite

    36. Multiplicative inverse of non-zero elements \(a\in Z\) is

      Ans: Non existent

    37. The \(\{x\in R, x^2 +10=0\}\) is

      Ans: Null set

    38. Additive identity on \(N\) is

      Ans: No identity

    39. Additive inverse of \(2+\sqrt{3}\) is

      Ans: \(-2-\sqrt{3}\)

    40. The multiplicative inverse of \(\sqrt{7}\) is

      Ans: \(\sqrt{7}/7\)
